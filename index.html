Case - VentureLabs

Desafio da VentureLabs que consiste da na criação de um formulário com mutiplas etapas. Link do deploy: https://case-stepform-spanol-95vxph4jx-spanol.vercel.app/clients As técnologias utilizadas foram: React, Css e axios.

Executar

Para executarmos o projeto precisamos:

  1. Clonar o repositório do projeto (seja pelo github desktop ou pelo gitbash).
  2. Abrir o a pasta com a clonagem do repositório utilizando o vscode (ou acessar a pasta pelo terminal do windows utilizando cd e o local da pasta).
  3. Já dentro da pasta devemos utilizar o comando npm install que será responsável por instalar as dependencias do app.
  4. Após isso vamos utilizar o comando npm start que inicializará nosso projeto.
  5. E com isso nosso projeto já estará rodando no localhost:3000
